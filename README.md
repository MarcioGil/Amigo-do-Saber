# Amigo do Saber

Plataforma educacional gratuita de reforço escolar com IA, gamificação e jogos educativos.

## Demonstração

Acesse a landing page demo com voz da Tia Dora:

[Demo Landing Page](https://marciogil.github.io/Amigo-do-Saber/tia-dora-landing.html)

> **Atenção:** Esta é uma versão de demonstração. A voz da Tia Dora está disponível apenas nesta página.

## Repositório

[Repositório no GitHub](https://github.com/MarcioGil/Amigo-do-Saber)

## Internacionalização

O projeto suporta múltiplos idiomas. Os arquivos de tradução estão em `frontend/locales/`:

- pt.json (Português)
- en.json (Inglês)
- es.json (Espanhol)
- ind.json (Indonésio)

## Como usar

1. Clone o repositório:

   ```sh
   git clone https://github.com/MarcioGil/Amigo-do-Saber.git
   ```

2. Abra o diretório `frontend` para acessar os arquivos da landing page.

3. Para visualizar a landing page demo com voz, acesse o link acima.
